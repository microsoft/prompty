**Prompty** is a file format (`.prompty`) for LLM prompts. Write your prompt
once — model config, inputs, tools, and template in a single markdown file —
then run it from Python, TypeScript, or C#. Treat prompts as code: version
them in git, test them in CI, share them across teams, and deploy with
confidence.

## Quick Example

```yaml
---
name: greeting
model:
  id: gpt-4o-mini
  provider: openai
  connection:
    kind: key
    apiKey: ${env:OPENAI_API_KEY}
inputSchema:
  properties:
    - name: userName
      kind: string
      default: World
---
system:
You are a friendly assistant.

user:
Say hello to {{userName}}.
```

<Tabs>
  <TabItem label="Python">
    ```python
    import prompty
    result = prompty.run("greeting.prompty", inputs={"userName": "Jane"})
    ```
  </TabItem>
  <TabItem label="TypeScript">
    ```typescript
    import { run } from "@prompty/core";
    const result = await run("greeting.prompty", { userName: "Jane" });
    ```
  </TabItem>
  <TabItem label="C#">
    ```csharp
    using Prompty.Core;
    var result = await Prompty.RunAsync("greeting.prompty", new() { ["userName"] = "Jane" });
    ```
  </TabItem>
</Tabs>

## Key Features

- **Three runtimes** — Python, TypeScript, and C#
- **Pipeline architecture** — render → parse → execute → process, each stage swappable
- **Built-in tracing** — console, JSON file, and OpenTelemetry backends
- **Provider support** — OpenAI, Azure / Foundry, and Anthropic
- **VS Code extension** — syntax highlighting, autocomplete, live preview, and trace viewer
